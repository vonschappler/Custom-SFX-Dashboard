About

Overview

  1. Used programming languages and frameworks:
    • Main script:
      • Python 2.7.3
    • Graphical User Interface (GUI):
      • Electron.js 13.1.4
      • Fomantic-UI 2.8.8
      • Noje.js 14.17.1
      • Datatables 1.10.25
      • HTML 5
      • CSS 3
      • JavsScript
      • JQuery 3.1.1
  2. Requirements:
    • Streamlabs Chatbot 1.0.2.64
    • Python 2.7.3
    • Operational System(s):
      • Windows 10
    • Streaming platform(s):
      • Twitch (100% tested)
    • Files to be used as sound effects have to be in MP3 format (Streamlabs Chatbot restriction)
  3. Basic functionalities:
    • Easily creation of custom sound effects
    • Import old settings from the script [Custom SFX v1.0.0] or start from scratch
    • Select any file on your computer to set as custom effect
    • Easy control of all defined sound effects
    and much more!

Changlelog:

    Oficial Releases:

    1. Custom SFX - v2.0.0
      2. - Added GUI for easy control of the sound effects
      Previously, the management of these sound effects was made using the Streamlabs Chatbot poor and limited GUI. Now, it's possible to manage these effects with a mordern and responsive GUI, with various controls.
      - Added some life quality features:
      • Added the possibility to specify any directory as base directory on your computer where sound effects are stored
        • Previously, only the script_root_folder/sfx was used for search of files to be added as custom sound effects. Now it's possible no only to still make use of the default folder (script_root_folder/sfx) but also select any folder to be set as default.
      • Added the possibility to select select any MP3 file from your computer as sound effect
        • Before, only MP3 files stored inside the script_root_folder/sfx could be selected. Now it's not only possible to change the default folder, but also to include files which are outside the default folder selected.
      • Added a function to live preview the selected file before creating a custom sound effect
        • Previously, due to the Streamlabs Chatbot limitations, previewing was possible, but it took too many steps/settings to work properly. Now a live preview can be done "on the fly" during the creation and edition of a custom sound effect.
      • Added a control panel to manage all added custom sounds effects easily
        • Before, due to the Streamlabs Chatbot limitations, editing / deleting a unwanted custom sound effect took to many steps which could be removed by creating a GUI.
      • Added an option to enable/disable a custom sound effect trigger
        • Before, the only way to prevent it from being triggered was by deleting it from the script.
      • Added functions to import backups created by the previous version
      • Added a logger function to log the script and GUI usage
        • It was not possible to log anything. Now with this logger, debug and error report will be easier to track and fix.
      • Added a improved help file
        • The previous help format has changed to a more intuitive format, with images and other features.
      - Added a functionatity to send important messages directly on the whisper of those who allow recieving whispers from non-friends
      As we know, not everyone allow receinving whispers, so the script now checks if the person allows whispers from non-friends or not and if the user accepts it, the script messages like errors and sound effects listing will be sent to the user whisper as a way to prevent spamming on stream chat.
      - Fixed and issue that would prevent sound effects listing to be sent if total lenght of message was over Twitch limit chat
      Due to IRC limitations, chat messages would either be sent incomplete or not sent at all.
    2. Custom SFX - v1.0.0
      3. - Added a command to list all sound effects triggers
      This feature was not available in Alpha versions
